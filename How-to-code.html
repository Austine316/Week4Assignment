Introduction to Coding

Coding can be defined as the act of writing a program instruction. The same way we can speak and write in different languages like English, French, German, etc is also the same way we can write program instructions in different programming languages like HTML, CSS, Javascript, Python, R, etc. The use of a programming language depends on what task it can perform. E.g, we can build websites with HTML, CSS and Javascript. There are more but in this article, we will focus on the basic use of HTML.

Coding with HTML

HTML (Hypertext Markup Language) is a standard markup language for building the skeleton of websites. We can style our website using the Cascading Style Sheets (CSS) and make it more dynamic and interactive using scripting languages such as JavaScript.

Basic HTML Tags

  1. Head Tag
  2. Header Tag
  3. Paragraph Tag
  4. Line break Tag

How to Use Major HTML Tags

Head Tag

The head tag contains metadata and title/information for the document. In html, the command line to call the head tag is <head> and we close it with </head>

Heading Tag

The heading tag defines the HTML headings. There are six different HTML headings (h1 to h6). These heading tags are different in sizes with h1 being the biggest and h6 being the smallest. In html, the command line to call the head tag is <h1> and we close it with </h1>. For other tags, the 1 is replaced with numbers from 2 to 6. E.g

<h1> This is an h1 Tag </h1>
<h2> This is an h2 Tag </h2>
<h3> This is an h3 Tag </h3>
<h4> This is an h4 Tag </h4>
<h5> This is an h5 Tag </h5>
<h6> This is an h6 Tag </h6>

Paragraph Tag

The Paragraph tag defines a new paragraph. In html, the command line to call the head tag is <p> and we close it with </p>

Line break Tag

The Line break tag breaks a line text and move to a new line. In html, the command line to call the head tag is <br>. The line break tag has no closing tag.
